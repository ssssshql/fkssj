# 疯狂水世界 - 辅助工具

基于 AutoJs6 的手游辅助工具，自动化操作"疯狂水世界"游戏。

## 功能

- **自动种植** — 收割 → 翻地 → 播种，库存满自动切换作物
- **自动锻造** — 定时检测锻造按钮并点击
- **搁浅商船** — 整点自动出征（单次/持续检测模式）
- **盟战** — 进攻/防守模式，自动勾选出征
- **兑换码** — 一键复制
- **悬浮球 + 控制面板** — 随时操控任务

## 运行

1. 将项目放到设备 `/sdcard/脚本/水世界/`
2. 在 AutoJs6 中打开 `main.js` 运行

## 项目结构

```
main.js                  # 入口：UI 布局、任务调度
modules/
  theme.js               # 颜色、作物配置、兑换码数据
  uiHelpers.js           # 日志、统计、导航、圆形图标
  floatingPanel.js       # 悬浮球 + 控制面板
  gameHelper.js          # 图像识别、资源加载
  plantFood.js           # 种植任务
  fleet.js               # 舰队通用操作
  war.js                 # 盟战任务
  merchantShip.js        # 搁浅商船任务
img/                     # 图像识别资源
test/                    # 测试脚本
```
